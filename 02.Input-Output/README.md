# Input and Output

Welcome to the **Input and Output** section of my Java Learning Journey.

This section covers the basics of reading input from the user and displaying output using Java.

## Topics Covered

1. Output using `System.out.print()`
2. Output using `System.out.println()`
3. Output using `System.out.printf()`
4. User input using `Scanner`
5. Reading different data types
6. Simple input and output programs

## Goal

Learn how to accept user input and display output effectively, which is the foundation for writing interactive Java programs.
